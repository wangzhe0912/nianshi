本系列博客主要讲解Python的web.py框架的使用。

web.py是一款轻量级的web开发框架，特点是简单易学且功能强大。


基本环境要求：

1. 安装python

直接从官网下载即可。

2. 安装web.py模块

可通过pip命令直接安装。在命令行状态下输入如下命令：

pip install web.py


一个最基本的web.py项目实例

新建一个demo.py文件，并编辑文件如下



<***script***> py
#-*- coding: UTF-8 -*-  

import web  

urls = (  
    '/', 'Index',  
)  
  
class Index(object):  
    def GET(self):  
        return "hello world!"  
  
if __name__ == '__main__':  
    app = web.application(urls, globals())  
    app.run()  
<***script_end***>
其中，urls用于指名url与对应操作的映射关系。

urls是一个元组，奇数项和偶数项分别表示url与操作类。

接下来，针对urls中对应的类，分别需要编写类来说明具体的操作。

针对每个类，可以针对get和post两种不同的请求方法定义不同的操作。

最后的部分表明运行该程序。


编辑好deno.py文件后，可以通过命令行执行python demo.py来启动web服务。

默认的端口号为8080。

打开浏览器，输入localhost:8080即可看到显示hello world!的网页。
